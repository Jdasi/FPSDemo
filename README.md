# Overview

A work in progress First-Person Shooter demo being developed in Unreal 5.7.3. This project is intended to explore gameplay systems such as Staged Weapon Reloads, Skinned Decals, Physical Animation, and Enemy AI. I will be prototyping the features primarily in Blueprint with the intention of refactoring as much as possible into C++ at a later time.

The goal is to produce a small gameplay loop where the player must fend off waves of enemies whilst managing their ammo reserves. The player will also have access to a melee attack which - besides serving as an alternate offensive option - will provide ample opportunity to showcase the Staged Weapon Reload system built into the player's weaponry.
